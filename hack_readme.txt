Project Title : Finding Hospitals

Start Date : 15-02-2024

End Date   : 29-02-2024
 

Project Description 

Module 1:

1. Navigate to Practo.com Home page.
2. Enter location  (like Chennai, Bangalore, Pune, etc).
3. Enter specialist (like Dentist, Cardiologist, etc).
4. Click on filters option.
5. Apply filters for Patient Stories,Experience, Fees, Availability and any one Sorting option.
6. Fetch First five doctors.
7. display the first 5 Doctor details.

Module 2:

1. Navigate to Practo.com Homepage.
2. Go to "Surgeries".
3. Extract all the surgeries in a List.
4. store all the surgeries in a List.
5. display them in console output.

Module 3:

1. Navigate to Practo.com Homepage.
2. click on "For Corporate" under that select Health and Wellness Plan .
3. fill the form with invalid email id & valid details for other fields.
4. validate if "Taking a demo" button is disabled.
5. Print the observation in console output.
6. Then provide email in valid format & click the demo button.
7. validate "Thank You" message is displayed.




Step to excute 

Testing      : click Testng.xml file then click on run the test.

Parallel     : click parallel.xml file then click on run the test.

crossbrowser : click CrossBrowser.xml file then click on run the test.
